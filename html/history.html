History

1989: Tim Berners-Lee proposes the idea of the WWW at CERN.

1990: Berners-Lee creates the first web browser, server and other fundamental web components to support the WWW.

1991: info.cern.ch, Berners-Lee's first website, is officially launched online.

1993: The graphical web browser Mosaic is launched, which boosts the use and popularity of the WWW.

1994: Berners-Lee founds the World Wide Web Consortium (W3C) to develop a web standard.
