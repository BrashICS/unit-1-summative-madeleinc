Inventions

World Wide Web (W3): A system designed to share and access information across the internet through documents/webpages.

Hyperlinks: The foundation of the W3, they're text and media that can be linked to make webpages easier to move across.

Hypertext Markup Language (HTML): A markup language that structures text, images and hyperlinks, for webpage structure

Hypertext Transfer Protocol (HTTP): A protocol that connects computers and servers to access and load webpages.

Uniform Resource Locator (URL): Unique addresses that each webpage has so that servers can locate information.
